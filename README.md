# Deluge
Deluge is a minimal weather app for all devices, including base features such as outlooks, radar, and more.

Deluge is currently in version BETA IV.

<h2>Version</h2>

The Current Version of this application is BETA IV.
Changelogs can be found in the "Releases" tab.

<h2>Creators and Contributors</h2>

<h3>About</h3>

- <b>The Repository:</b>

Deluge was first imagined in August of 2025, being inspired by the <a href="https://github.com/SteepAtticStairs/AtticRadar">AtticRadar</a> repository made by GitHub User <a href="https://github.com/SteepAtticStairs">StepAtticStairs</a>. I (JesseWx2011) wanted to create an app fully minimal and fully free, with no ads, and just a basic experience viewing active alerts, radar data, and outlooks. Just the essentials.

This repository was created on June 22, 2026, and is, as of now, awaiting a full Public release with all the features. This repository will continually be managed by its creators and contributors.

- <b>The Creator</b>

JesseWx2011 (Jesse Hasty) is a young content creator, programmer in Python, HTML, JS, and CSS, and a basic weather enthusiast from Florida. Jesse has been coding from age 12 and loves creating weather-related projects and content on his <a href="https://www.youtube.com/@JesseLikesWeather">YouTube Channel</a>.
